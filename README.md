# 水晶能量測驗

這是一個可以直接發布到 GitHub Pages 的靜態網站。主要檔案：

- `index.html`：網站頁面
- `styles.css`：視覺樣式
- `script.js`：測驗互動與結果

## 發布到 GitHub Pages

1. 在 GitHub 建立一個新的 repository。
2. 把這個資料夾裡的檔案上傳到 repository 根目錄。
3. 進入 repository 的 `Settings`。
4. 到 `Pages`，在 `Build and deployment` 選 `Deploy from a branch`。
5. Branch 選 `main`，資料夾選 `/(root)`，按 `Save`。
6. 等待 GitHub Pages 產生網址。

官方說明可參考 [GitHub Pages publishing source](https://docs.github.com/en/pages/getting-started-with-github-pages/configuring-a-publishing-source-for-your-github-pages-site)。
